# Proyecto: Conexión con Google Gemini API

Este proyecto demuestra cómo conectarse a la API de Google Gemini usando Python y un entorno virtual en VS Code.

Requisitos previos
* Python 3.10 o superior
* VS Code instalado
* Git instalado
* Cuenta en Google AI Studio
* Clave API de Google Gemini

 Instalación y configuración

1️. Clonar el repositorio

```bash
git clone https://github.com/Esteban812112/gemini-api-curso.git
cd gemini-api-curso
```

2️. Crear entorno virtual

En Windows:

```bash
python -m venv env
.\env\Scripts\activate
```

3️. Instalar dependencias

```bash
pip install -r requirements.txt
```

4️. Configurar la API Key

Crea un archivo `.env` en la raíz del proyecto con este contenido:

```
GEMINI_API_KEY=TU_CLAVE_AQUI
```

5️. Ejecutar el script

```bash
python app_gemini.py
```

Si todo está correcto, verás una respuesta generada por Gemini en la terminal.
